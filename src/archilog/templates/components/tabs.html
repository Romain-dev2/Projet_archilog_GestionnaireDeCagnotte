{# components/tabs.html ───────────────────────────────────────────────────────────────────────── Macros de navigation par onglets. Importées via macros.html. Deux macros exposées : • tabs_cagnotte(nom, active) — onglets d'une cagnotte (Vue d'ensemble, Dépenses, Équilibre, Ajouter) • tabs_home(active) — onglets de la page d'accueil (Cagnottes, Nouvelle cagnotte) Paramètre `active` : chaîne correspondant à la valeur de l'onglet courant (ex : 'depenses', 'overview'). Détermine la classe CSS .active et l'attribut aria-selected pour l'accessibilité. Accessibilité : - role="tablist" sur le conteneur. - role="tab" + aria-selected sur chaque onglet. - Les icônes SVG sont aria-hidden="true" (décoration, texte adjacent suffit). - L'indicateur actif est géré visuellement par CSS (border-bottom) ET sémantiquement par aria-selected="true". Note : les onglets sont des (liens) et non des
